2026 결과보고서

1. 과제명

CCTV 기반 도로 위험 요소 자동 인식 시스템 구축

2. 수행 목적

본 과제는 도로에 설치된 CCTV 또는 PC에 연결된 웹캠 영상을 활용하여 도로 위험 요소를 자동으로 탐지하고, 감지 결과를 대시보드에 시각화하는 것을 목표로 한다. 주요 탐지 대상은 사슴과 같은 야생동물, 도로 위 쓰레기, 도로 크랙이며, 향후 YOLO 기반 객체 탐지 모델을 학습하여 실제 도로 환경에서 활용 가능한 인공지능 안전 모니터링 시스템으로 확장하는 것을 목적으로 한다.

3. 추진 배경 및 필요성

도로 위의 야생동물 출현, 낙하물 및 쓰레기, 도로 표면 균열은 운전자 안전에 직접적인 영향을 미친다. 이러한 위험 요소는 즉시 발견하고 관리 기관 또는 운전자에게 전달하는 것이 중요하다. 기존에는 사람이 직접 CCTV 화면을 확인하거나 민원 접수를 통해 위험 상황을 파악하는 방식이 많았으나, 실시간성이 부족하고 반복적인 모니터링 부담이 크다.

4. 주요 개발 내용

4.1 CCTV 자동 연결 기능

브라우저 카메라 권한 문제를 해결하기 위해, 웹브라우저가 직접 카메라에 접근하는 방식 대신 Python 서버가 OpenCV를 이용해 CCTV 카메라를 직접 연결하도록 구현하였다.

4.2 CCTV 프리뷰 제공

서버에서 수집한 카메라 프레임을 JPEG 이미지로 변환하여 대시보드에 표시하도록 구현하였다.

4.3 위험 요소 감지 기능

  • 도로 크랙: 엣지 및 선분 검출 기반 감지
  • 도로 쓰레기: 색상 및 객체 후보 영역 기반 감지
  • 사슴 후보: 갈색 계열 동물형 영역 기반 감지
  • 포트홀 후보: 어두운 도로 표면 후보 영역 기반 감지

4.4 감지 영역 사각형 표시

탐지된 객체 또는 위험 요소의 위치를 bounding box 좌표로 계산하고, CCTV 프리뷰 이미지 위에 사각형과 라벨을 표시하도록 구현하였다.

4.5 YOLO 학습 구조 준비

YOLO 기반 객체 탐지 모델을 학습할 수 있도록 데이터셋 구조, 학습 스크립트, 모델 적용 경로를 구성하였다.

  • 클래스: deer, trash, crack
  • 데이터셋 설정: datasets/cctv_yolo/data.yaml
  • 학습 스크립트: scripts/train_yolo.py
  • 모델 저장 위치: models/cctv_best.pt

5. 시스템 구성

  1. CCTV 또는 웹캠이 PC에 연결된다.
  2. Python 서버가 OpenCV로 카메라를 자동 연결한다.
  3. 서버가 프레임을 수집하고 위험 요소를 분석한다.
  4. 분석 결과를 이벤트로 저장한다.
  5. CCTV 프리뷰 이미지에 감지 박스와 라벨을 표시한다.
  6. 대시보드에서 감지 결과와 지도 정보를 확인한다.

6. 구현 결과

구현된 대시보드에서는 CCTV Detection 영역을 통해 카메라 프리뷰를 확인할 수 있으며, 감지된 위험 요소가 프리뷰 이미지 위에 사각형으로 표시된다. Live Hazard Alerts 영역에는 감지 이벤트가 시간순으로 누적 표시된다.

7. 한계점

현재 OpenCV 기반 감지는 색상, 엣지, 모양 등의 조건을 이용한 휴리스틱 방식이므로 실제 도로 환경에서는 오탐과 미탐이 발생할 수 있다. 특히 사슴과 같은 동물 객체는 YOLO와 같은 학습 기반 객체 탐지 모델이 필요하다.

8. 향후 개선 계획

  1. 실제 도로 CCTV 이미지 수집
  2. deer, trash, crack 데이터 라벨링
  3. YOLO 모델 학습 및 성능 평가
  4. 학습된 모델을 models/cctv_best.pt로 적용
  5. Google Cloud 또는 AWS GPU 서버를 활용한 학습 환경 구축

9. 기대 효과

본 시스템은 CCTV 기반 도로 위험 요소 자동 감지를 통해 도로 안전 관리의 효율성을 높일 수 있다. 향후 YOLO 모델을 적용하면 실제 도로 환경에서 사슴, 쓰레기, 도로 크랙을 더욱 정확하게 탐지할 수 있을 것으로 기대된다.

10. 결론

본 과제를 통해 CCTV 카메라 자동 연결, 프리뷰 표시, 위험 요소 감지, 사각형 표시, 이벤트 저장 및 대시보드 시각화 기능을 구현하였다. 현재는 OpenCV 기반 프로토타입 단계이지만, YOLO 학습 구조를 함께 마련하여 실제 인공지능 객체 탐지 시스템으로 확장할 수 있는 기반을 구축하였다.
